1. Tổng quan dự án

VinhKhanhTourGuide là hệ thống thuyết minh du lịch thông minh hoạt động theo mô hình Fullstack Client-Server. Hệ thống cho phép quản lý tập trung nội dung các điểm đến và tự động cập nhật đến thiết bị của khách du lịch thông qua API.

2. Kiến trúc 3 Lớp (Three-Tier)

  • Presentation Layer: .NET MAUI (Mobile) & ASP.NET Core MVC (Web Admin).
  • Application Layer (API): Web API RESTful cung cấp dữ liệu qua cổng 5099.
  • Data Layer: SQL Server (Trung tâm) & SQLite (Cache Offline).

3. Phân hệ Web Admin (MVC)

Xây dựng trên ASP.NET Core MVC giúp quản trị viên điều hành toàn bộ dữ liệu:

  • Cơ chế Localization: Ép chuẩn số en-US để lưu tọa độ GPS chính xác với dấu chấm thập phân.
  • Quản lý POI: Giao diện CRUD (Thêm/Sửa/Xóa) quán ăn trực tiếp vào SQL Server trung tâm.

4. Phân hệ Web API (Cầu nối)

Thông số kỹ thuật API:
- Endpoint: /api/pois
- IP Máy ảo: 10.0.2.2
- Cấu hình: usesCleartextTraffic="true" (Cho phép kết nối HTTP nội bộ).

5. Phân hệ Mobile App (MAUI)

5.1. Cơ chế Radar & Geofencing

Sử dụng IDispatcherTimer quét vị trí mỗi 5 giây. Nếu khoảng cách <= GeofenceRadius, hệ thống kích hoạt thuyết minh.

5.2. Offline-First Logic

Dữ liệu từ API được đồng bộ và ghi đè vào SQLite Local mỗi khi App có mạng, đảm bảo thuyết minh vẫn hoạt động ở những khu vực sóng yếu.

6. Thiết kế Cơ sở dữ liệu (Chi tiết)

6.1. Bảng Điểm đến (POI Table)

7. Quy trình Xử lý Thuyết minh

1. Quét vị trí GPS -> So sánh với GeofenceRadius.
2. Kiểm tra SQLite bảng TranslationCache (theo PoiId + Ngôn ngữ máy).
3. Nếu chưa có: Gọi Gemini AI dịch Description_VN -> Lưu Cache.
4. Phát TTS (Text-to-Speech) và Vẽ vòng tròn Highlight đỏ trên bản đồ.

8. Sơ đồ tuần tự (Sequence Diagram)

sequenceDiagram
    participant AD as Web Admin (MVC)
    participant SQL as SQL Server
    participant API as Web API
    participant APP as Mobile App (MAUI)
    participant AI as Gemini AI

    AD->>SQL: Lưu bài review & GeofenceRadius (Dấu chấm thập phân)
    APP->>API: GET /api/pois (Đồng bộ dữ liệu)
    API->>SQL: Truy vấn POIs
    SQL-->>API: Trả về POIs
    API-->>APP: Trả về JSON (Port 5099)
    APP->>APP: Lưu vào SQLite & Quét Radar GPS
    Note over APP, AI: Vào vùng Geofence (30m)
    APP->>AI: Dịch Description_VN (Nếu Cache trống)
    AI-->>APP: Trả về bản dịch
    APP->>APP: Phát thuyết minh & Vẽ vòng tròn đỏ
